Como atualizar os produtos no site

  1. Preencha o formulário abaixo com os dados do novo produto.
  2. Clique em "Baixar produtos.json" para salvar o arquivo atualizado.
  3. Acesse o repositório onde está localizado o produtos.json.
  4. Substitua o arquivo produtos.json pelo novo arquivo baixado.

Dica: Você pode editar, remover ou adicionar quantos produtos quiser antes de baixar o arquivo.

Dica: Deixe a URL da imagem em branco se ainda não tiver imagem do produto.
