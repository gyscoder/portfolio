Projetos

Landing Page Lei da Atração

Landing para curso fictício sobre a Lei da Atração, com design inspirador, formulário de inscrição, vídeo de apresentação e carrossel de depoimentos. Projetada para conversão, com identidade visual suave e foco em despertar o interesse do usuário desde o primeiro contato.

HTML, TailwindCSS, JavaScript / React

App de Finanças Pessoais

App simples para controle financeiro pessoal, com cadastro de receitas e despesas, cálculo automático do saldo, histórico de transações e remoção de entradas. Além disso, mantém persistência dos dados no navegador via localStorage e gera um log das ações para download.

HTML, CSS (TailwindCSS + arquivo externo), JavaScript (DOM, localStorage, manipulação dinâmica)

To-Do List App

Aplicativo simples para organização de tarefas diárias com suporte a filtros, marcação de tarefas concluídas, remoção e persistência local. Permite também gerar e baixar um log completo das ações realizadas.

HTML, TailwindCSS, JavaScript (DOM, localStorage)

Mini E-commerce de Produtos

Loja virtual simples com produtos carregados via JSON e layout responsivo em TailwindCSS. Cada item possui link direto para WhatsApp com mensagem personalizada, ideal para vendas informais e artesanais.

HTML, TailwindCSS, JavaScript (fetch, JSON, integração WhatsApp)
